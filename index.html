Mathew Mariani / Interactive Demos

A collection of small interative demos.

These demos are built using C++ compiled to WebAssembly, with VueJS for interfacing, and statically-generated using Astro.

Maze Generator

Various maze generation algorithms.

Chess

A chess engine.

Dungeon Generator

Zelda-like dungeon generator inspired by the Binding of Isaac.

John Conway's Life

Cellular automata.

Minesweeper

Classic Minesweeper.

TicTacToe Bot

An implementation of the MiniMax algorithm.

Pathfinding

Various pathfinding algorithms.
